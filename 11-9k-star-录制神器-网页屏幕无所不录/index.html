录制神器 !!! 网页,屏幕无所不录

  • 哟喂
  • 科技资讯

软件介绍

Screenity是一款高效、易用的屏幕录制工具，它不仅能够帮助用户捕捉屏幕上的每一个细节，还能够在录制过程中添加各种标注，使得视频内容更加生动和直观。这款软件特别适合需要制作教学视频、产品演示、会议记录或任何需要屏幕录制和标注的场景。它适用于桌面上的Chrome和基于Chromium的浏览器

特点

Screenity的特点是无限制的录制，不限时长、不限数量以及不限区域的视频，支持离线操作，不收集任何数据。它提供了声音录制选项，可录制麦克风或内部音频，并使用按键对讲等功能。视频上，AI技术提供的相机背景或模糊效果以增强录像。在录制中，用户可以在屏幕上任意位置进行绘图批注，添加文本、箭头、形状等元素；突出显示光标和点击操作，以及聚焦模式；聚焦特定区域的平滑缩放；对页面上的敏感内容的模糊处理等。此外，它还提供了全面的编辑器，支持导出为mp4、gif和webm格式，或直接保存到Google Drive并分享链接视频文件。

主要功能

  • 录制整个标签页、特定区域、桌面、任何应用程序或摄像头。
  • 独立的麦克风和电脑音频控制。
  • 在屏幕上的任何地方绘图、添加文本和创建箭头来进行注释。
  • 突出显示鼠标点击，聚焦鼠标，或从记录中隐藏它。
  • 设置倒计时、隐藏部分UI或将其移动到任意位置。
  • 导出为mp4、gif和webm格式，或保存到Google Drive。

安装指南

从应用商店安装

  • 访问Chrome网上应用店。
  • 搜索“Screenity”。
  • 点击“添加到Chrome”按钮。
  • 确认添加，Screenity即安装完成。

源码安装

  • 访问Screenity的GitHub页面。
  • 下载源码或使用Git命令克隆仓库到本地：

    git clone https://github.com/alyssaxuu/screenity.git
  • 解压下载的文件（如果需要）。
  • 打开Chrome浏览器，进入chrome://extensions/页面。
  • 开启右上角的“开发者模式”。
  • 点击“加载已解压的扩展程序”，选择包含Screenity源码的文件夹。

Screenity是一款免费、开源的屏幕录制工具，它提供了丰富的录制和标注功能，非常适合需要制作教学视频、产品演示或进行屏幕分享的用户。它的安装和使用都非常简便，且不限制录制时长，是一个强大的屏幕录制利器。
